# CSU Systems Engineering - Conference Poster Template

A LaTeX poster template for the Department of Systems Engineering at Colorado State University, based on the `tikzposter` package. Designed for 24 x 36 inch conference posters.

## Overleaf

View and edit on Overleaf: [https://www.overleaf.com/read/txhjfvqfydfn#8bcf7c](https://www.overleaf.com/read/txhjfvqfydfn#8bcf7c)

## Files

- `main.tex` - Main poster document
- `csu-theme.sty` - CSU color theme and styling
- `refs.bib` - Bibliography file
- `csu.pdf` - CSU logo
- `CSU_bottom_left.png` - Systems Engineering department logo
- `CSU_bottom_center.png` - QR code
- `CSU_bottom_right_green-yellow_bar.png` - Bottom banner
- `CSU_template_draft_scan_QR.png` - Draft/reference QR code
